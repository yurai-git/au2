# AU2

A package to create directories for AviUtl 2 projects, completely for private
use.

## Installation

```bash
npm install -g @yurai/au2
```

```bash
yarn global add @yurai/au2
```

```bash
pnpm add -g @yurai/au2
```

```bash
bun add -g @yurai/au2
```

## Commands

### `au2`

Creates directories.

#### Options

##### `--version`, `-v`

Output the version number and exit.
